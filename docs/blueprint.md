# **App Name**: Fiscal Flow

## Core Features:

- Transaction Input: Add transactions with details such as date, amount, and category. Includes options for recurring transactions.
- Category Management: Categorize transactions for detailed tracking and analysis. Predefined categories plus customization.
- Budget Setting: Set monthly budgets for different categories. Tool warns if over budget, predicts expenses to recommend adjusting budget before getting there.
- Spending Visualization: Visualize spending patterns using interactive charts, graphs and dashboards.
- Dashboard Overview: View key financial insights, budget status, and recent transactions on a personalized dashboard.
- Report Generation: Create visual reports on your budgeting with comparisons over periods of time.

## Style Guidelines:

- Primary color: Soft blue (#A0C4FF) to evoke trust and clarity.
- Background color: Off-white (#F0F4F8) to maintain a clean and professional aesthetic.
- Accent color: Darker blue (#75A3FF) for interactive elements and emphasis.
- Body and headline font: 'Inter' (sans-serif) for readability and a modern look.
- Use clear and intuitive icons to represent categories and actions.
- Maintain a clean, grid-based layout with ample white space.
- Subtle transitions and animations to provide feedback and enhance user experience.
